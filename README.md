# GoodWave

GoodWave is a modern nonprofit website built for the Next.js UI/UX Hackathon concept. It is designed to help users discover meaningful social programs, understand real impact, and support causes through a clean donation experience.

## Tagline

**Every Donation Creates a Future**

## Project Overview

GoodWave is a futuristic nonprofit web platform focused on social impact. The website presents programs, impact statistics, donation options, and nonprofit storytelling in a clean and attractive UI.

The goal of this project is to create a user-friendly nonprofit website experience where visitors can:

- Understand the mission quickly
- Explore social impact programs
- View program details
- Donate easily
- See transparent impact information
- Navigate through a clean and responsive interface

## Features

- Modern nonprofit landing page
- Programs discovery page
- Program detail page
- Donation flow page
- Impact page
- Responsive navigation
- Dark premium UI design
- Program cards
- Impact statistics
- Donation call-to-action sections
- Shared styling with reusable CSS
- Simple JavaScript navigation support

## Pages

- `index.html` - Home page
- `programs.html` - Programs listing page
- `program-detail.html` - Program details page
- `donate.html` - Donation page
- `impact.html` - Impact / About page

## Tech Stack

- HTML
- CSS
- JavaScript
- Tailwind CSS
